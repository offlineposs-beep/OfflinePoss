# **App Name**: Offline Repair POS

## Core Features:

- Product Catalog Management: Add, edit, and delete product inventory items with details like name, category, SKU, cost, retail price, and stock level.
- Offline Sales Transactions: Process sales transactions entirely offline, including product selection, pricing calculations, discounts, and payment recording.
- Repair Job Tracking: Log and track repair jobs with customer details, device information, issue description, and status updates.
- Offline Reporting: Generate basic sales reports (daily/weekly) and view transaction logs, stored offline.
- Low Stock Alerts: Generate alerts when a product drops below a preset threshold
- Receipt Generation: Generate and store a sales receipt viewable as PDF or image.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) for trustworthiness and reliability.
- Background color: Light gray (#E5E7EB), offering a neutral, clean backdrop.
- Accent color: Teal (#306979) for interactive elements and key actions.
- Body and headline font: 'PT Sans', a humanist sans-serif that provides a modern and slightly warm feel suitable for both headings and body text.
- Use clear, consistent icons to represent inventory items, repair statuses, and payment methods.
- Maintain a clean, intuitive layout with clear section divisions to facilitate quick access to critical information.
- Use subtle animations to indicate changes in stock level or repair status.
